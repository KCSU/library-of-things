Site Manual

Everything that you (hopefully!) need to know to manage the Library of Things website.

0 Getting Started

Since you are able to see this page, you must already be an administrator and should have access to the admin sidebar on the left. To assign administrator privileges to other users, please contact the computing officer (for now).

Under "Item Requests", you will see all pending requests from users to borrow items. For each loan, you can either confirm item delivery (which will start the timer for the loan) or refuse the request.

Under "Manage Inventory", you will see a list of all items. You can add new items, and also delete and edit existing items. Notably, you can also hide items, which makes them invisible to regular users but still visible in the admin inventory.

For each item, you can choose from a pre-determined set of loan durations (7 days / 30 days / end of academic year / permanent). This determines how long an item can be borrowed for. "End of academic year" here means the next 30th June, while "permanent" here means the item will be given away, permanently.

You can also put out public announcements that will be displayed on the homepage for all users to see. You can do this in the "Settings" section.

3 How do I upload <image>?

You should have edit access to this Google Drive folder. If not, please obtain access from the computing officer.

Upload the image to the folder, right-click on it, and select "Copy Link". The image should be publicly viewable. The link should be of the form 

https://drive.google.com/file/d/<image_id>/view[?...]

Copy the image ID, then paste it inside the following URL template:

https://lh3.googleusercontent.com/d/<image_id>

You can then use this URL as the image URL when adding or editing items in the inventory.

This is a temporary solution. If this proves to be too much of a hassle, Lucas is open to suggestions for improvement.
